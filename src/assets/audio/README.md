# E-Darshan Mandir - Audio Assets

This directory contains devotional audio files for the temple application.

## Required Audio Files

Please add the following audio files to this directory:

1. **hanuman-chalisa.mp3** - Hanuman Chalisa recitation
2. **hanuman-aarti.mp3** - Hanuman Aarti
3. **om-chant.mp3** - Om chanting

## Where to Get Audio Files

You can source royalty-free devotional audio from:
- YouTube Audio Library (ensure you have proper licenses)
- Free music archives
- Record your own chants
- Use text-to-speech for mantras

## Audio Format Recommendations

- **Format**: MP3 or OGG
- **Bitrate**: 128 kbps (balance between quality and file size)
- **Sample Rate**: 44.1 kHz
- **Duration**: 3-5 minutes recommended

## Legal Notice

Ensure you have the rights to use any audio files you add here. This app is for devotional purposes only.

## Fallback

If audio files are not available, the audio player will show an error message and users can still use other features of the app.
