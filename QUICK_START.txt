Minimal Usage (No UI):
──────────────────────────────────────────────────────────────────
import { AmbientAudioService } from './services/ambient-audio.service';

constructor(private ambient: AmbientAudioService) {}

onClick() {
  this.ambient.initialize();  // Required first!
  this.ambient.ringBell();
  this.ambient.playShankh();
}

With Observable:
──────────────────────────────────────────────────────────────────
this.ambient.isTempleOpen$.subscribe(isOpen => {
  console.log(isOpen ? 'Temple Open' : 'Temple Closed');
});

Audio doesn't play:
  ✓ Ensure initialize() called from user gesture
  ✓ Check browser console for errors
  ✓ Verify audio files exist in src/assets/audio/
  ✓ Check if muted

Ambience doesn't auto-start:
  ✓ Check current time (must be 5 AM - 7 PM)
  ✓ Ensure initialize() was called
  ✓ Check mute state
  ✓ Console.log isTempleOpen$ observable

Volume too low:
  ✓ Use setVolume(1.0) for max
  ✓ Check device/browser volume
  ✓ Verify audio files not corrupted
